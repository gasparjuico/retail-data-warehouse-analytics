# Retail Data Warehouse & Analytics Project

## Overview

An end-to-end analytics project that transforms raw retail CRM and ERP data into a structured data warehouse for reporting and dashboarding.

This project demonstrates practical skills in:

- SQL Server
- ETL / Data Cleaning
- Data Warehouse Design
- Data Modeling
- Business Analysis
- Power BI Dashboarding
- Git / GitHub

---

## Business Goals

This project answers key business questions:

1. How is revenue trending over time?
2. Which products generate the most sales?
3. Which product categories perform best?
4. Which countries or regions lead revenue?
5. Who are the most valuable customers?
6. How can customers be segmented?

---

## Dataset

Simulated retail company data combining CRM and ERP source systems.

Includes:

- Customer records
- Product catalog
- Sales transactions
- Regional / location data
