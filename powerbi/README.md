# Power BI Dashboard

This folder contains the final dashboard layer of the Retail Data Warehouse & Analytics Project.

## File Included

- `retail_analytics_dashboard.pbix`

## Dashboard Pages

### Executive Dashboard
High-level KPIs, revenue trends, and overall performance.

### Customer Dashboard
Customer rankings, segmentation, and value analysis.

### Product Dashboard
Product and category performance analysis.

### Regional Performance Dashboard
Revenue distribution by country.

## Objective

Transform warehouse and SQL insights into interactive business intelligence dashboards for decision-makers.
