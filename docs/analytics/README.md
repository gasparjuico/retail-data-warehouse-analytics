# Analytics Documentation

This folder contains visual assets and supporting materials for the SQL analytics section of the project.

## Purpose

These documents help communicate key business insights generated from the retail data warehouse using SQL analysis.

## Folder Structure

### roadmap/
Contains the analytics workflow and analysis roadmap.

Files:
- `analytics_roadmap.png`

### screenshots/
Contains charts, screenshots, and result visuals produced from SQL queries.

Files include:
- `measures_summary.png`
- `revenue_over_time.png`
- `top_10_customers_by_revenue.png`
- `top_10_products_by_revenue.png`

## Key Insights Covered

- Revenue trends over time
- KPI and measure summaries
- Top-performing customers
- Top-performing products

## Related Sections

- `sql/analytics/` → SQL scripts and CSV outputs
- `powerbi/` → Interactive dashboard layer
- `docs/dashboard/` → Dashboard screenshots and visuals

## Objective

Translate raw SQL outputs into clear, business-friendly visuals for decision-making and portfolio presentation.
