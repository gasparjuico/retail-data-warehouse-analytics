# Analytics SQL

This folder contains SQL scripts and outputs used to analyze the retail data warehouse and generate business insights.

## Structure

### scripts/
SQL queries for exploration, KPI generation, segmentation, and reporting.

### outputs/
CSV exports generated from SQL query results.

## Analysis Categories

### Exploration
- Database structure review
- Dimension table exploration
- Date range exploration
- Core measures exploration
- Magnitude exploration
- Ranking

### Advanced Analytics
- Revenue trends over time
- Cumulative revenue
- Period-over-period change
- Part-to-whole analysis
- Customer segmentation

### Reporting
- Customer performance report
- Product performance report

## Objective

Transform warehouse data into actionable business insights for decision-making.
